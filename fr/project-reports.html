Rapports générés par Maven

Ce document fournit une vue d'ensemble des divers rapports générés automatiquement par Maven . Chaque rapport est décrit brièvement ci-dessous.

Vue d'ensemble

Document Description
Changements Rapport des changements dans les versions publiées du projet.
Liste des tags Rapport sur les occurrences de tags trouvés dans le code.
Property Updates Report Provides details of properties which control versions of dependencies and/or plugins, and indicates any newer versions which are available.
Dependency Updates Report Provides details of the dependencies which have updated versions available.
Plugin Updates Report Provides details of the plugins used by this project which have newer versions available.
FindBugs Génère un rapport du code source avec la librairie FindBugs.
Checkstyle Rapport sur les conventions de style de codage
JavaDocs JavaDoc API documentation.
Sources Xref Code source Java généré au format HTML.
